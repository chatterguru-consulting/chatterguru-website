Model Flow

1Configure waveform, radar, target, clutter, and platform motion.
2Estimate target return strength with range-dependent scaling.
3Add synthetic land, sea, rain, snow, and receiver noise components.
4Apply motion-compensated apparent Doppler when selected.
5Render a range-by-radial-velocity intensity map.

This browser dashboard mirrors the repository model flow for fast scenario exploration. It uses fictional, synthetic, unclassified values and display scaling only (not a physics-validated model); use the Python signal chain for tested waveform-driven runs.
